Headers → Tags

Each markdown header becomes a lowercase XML tag. Spaces and special characters are replaced with underscores.

# Main Title → <main_title>

Italic/Bold Headings

Lines with italic (*Section* or _Section_) or bold (__SECTION__) formatting at the root level (no indentation) are treated as level 1 headers.

*Context*
__FORMAT__
__EXAMPLE__

becomes:

<context>
<format>
<example>

Nesting

Smaller headers are nested inside larger ones based on heading level.

# Title
## Subtitle

becomes:

<title>
  <subtitle>
  </subtitle>
</title>

Lists

Bullet lists (- item) become plain <item> tags.

Numbered lists (1. item) include a number attribute.

- Bullet item
1. Numbered item

becomes:

<item>Bullet item</item>
<item number="1">Numbered item</item>

Code Blocks

Fenced code blocks are wrapped in <code lang="..."> tags with CDATA.

```js
console.log("hello");
```

becomes:

<code lang="js"><![CDATA[
console.log("hello");
]]></code>
